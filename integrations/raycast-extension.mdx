## Overview

The Compresto Raycast extension enables file compression directly from Raycast through integration with the Compresto application.

<Note>
Compresto macOS app must be installed on your system for the extension to function properly.
</Note>

## Installation

Install the extension from the [Raycast Store](https://www.raycast.com/hieudinh/compressx).

## Available Commands

| Command | Description |
|---------|-------------|
| **Compress Files** | Compress selected files with Compresto |
| **Import Files** | Import selected files into Compresto |

## Features

- Compress multiple files simultaneously
- Import files directly into Compresto for processing
- Significant file size reduction capabilities
- Seamless Raycast integration
