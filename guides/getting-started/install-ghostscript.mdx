---
title: "Install Ghostscript"
description: ""
---

To compress PDFs in Compresto, you must have [Ghostscript](https://www.ghostscript.com/) installed. Developed by Artifex Software, Inc., Ghostscript is the leading PDL conversion and interpreter tool, providing native PDF rendering. 

If you have Homebrew installed, you can simply press the "Install Ghostcript via Homebrew" button.

![install_ghostscript](/images/install_ghostscript.png)

If you don't have Homebrew installed, follow one of these methods below to install Ghostscript

<Tabs>
  <Tab title="Method 1: Install via Homebrew">
    <Steps>
      <Step title="Install Homebrew">
        https://brew.sh/
      </Step>
      <Step title="Install Ghostscript via Homebrew">
        https://formulae.brew.sh/formula/ghostscript
      </Step>
    </Steps>
  </Tab>
  <Tab title="Method 2: Install via MacPorts">
    <Steps>
      <Step title="Install MacPorts">
        https://www.macports.org/install.php
      </Step>
      <Step title="Install Ghostscript via MacPorts">
        https://ports.macports.org/port/ghostscript/
      </Step>
    </Steps>
  </Tab>
</Tabs>

<Steps>
  <Step stepNumber={3} title="Once the installation is done, open the Terminal app and enter `which gs`">
    You will see the path of Ghostscript printed as the output

    ```
    $ which gs
    > /opt/local/bin/gs
    ```

    ![ghostscript path](/images/ghostscript_path.png)
    <Note>The output of the `which gs` command may vary depending on the installation method used.</Note>
  </Step>
  <Step stepNumber={4} title="Then copy the output and set it as the path for Ghostscript">
    ![ghostscript path](/images/update_ghostscript_path.png)
  </Step>
  <Step stepNumber={5} title="You're done">
    You can now compressing your PDF
  </Step>
</Steps>
